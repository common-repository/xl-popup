XL PopUp is a responsive WordPress plugin for display popup dialog.

== Description ==

XL PopUp is a WordPress plugin which create popup box / form. It is
simple but flexible & powerful. No need any coading knowledge. Just install and it will display popup one time per day.

Plugin Features

* Plug and play
* No Shortcode
* Work with any kind of form.
* Contact form 7 or Gravity forms shortcode support.
* Work with all Wordpress theme.
* Easy to use.
* Developer friendly & easy to customize.
* Its easy user interface allows you to add form code or any shortcode etc.


== Installation ==

Install XL PopUp as a regular WordPress plugin. Here is different ways to install XL PopUp plugin :


= Install by Search Plugin =
- In your Admin, go to menu Plugins > Add New
- Search XL PopUp
- Click to install
- Activate the plugin
- XL PopUp Setting page will show Settings > XL PopUp

= Install by Upload Plugin =
- Download the latest version of XL PopUp (.zip file)
- In your Admin, go to menu Plugins > Add New
- Select  "Upload Plugin"
- Click on “Choose File”
- Select downloaded xl-popup.zip & click on ‘Install Now’ button
- Activate the plugin
- XL PopUp Setting page will show Settings > XL PopUp

= Install Plugin using FTP =
- Upload xl-popup folder inside the /wp-content/plugins/ directory
- Go to WordPress dashboard > Plugins & Activate the XL PopUp
- XL PopUp Setting page will show Settings > XL PopUp


== Frequently Asked Questions ==

= How do I add PopUp Form? =
Very easy to add popup form. Just install popup plugin and go to XL PopUp menu under Settings. Add your form code
here and save.

= Where do I get form code? =
If you have mailchimp/infusionsoft account, then copy subscription form code from your account.
